JQuery Mobile App Development

I have been using JQuery Mobile for about a year now so I thought I would write up my thoughts on it. I also will include some useful add-ins I have used in conjunction to JQuery Mobile in my development travels.

Reasons to go there

Prototypes: Great for quick prototypes.

Because it is so quick to get started with a JQuery Mobile project I will often use it to create quick prototypes around a concept I am working on. While the final product may end out being created with a different technology if you need to quickly get something running to help visualize an idea then JQuery Mobile will defiantly be your friend.

Ease of learning: Easy to get started and understand.

As long as you know a bit of simple html and can follow the setup instructions given you can get rolling fairly quickly with this simple framework. The docs are relatively easy to follow and is a good deal of ideas flowing in place like stackoverflow. Oh and there is the provided theme roller to really help get things moving.

Reasons not to go there

Your a control freak: ok maybe a little strong :)

Using the JQuery Mobile framework things can begin to all look a bit the same, and if you what fine control over the look of your app you would be better served looking else where rather than fight with it.

You want full interaction with the phone:

This obviously goes for all web apps but there comes a point where if you want to do certain things with the users phone you need to either move to a native app for the target phones or use a technology such as PhoneGap that will convert your web app to a native app.

Projects I found useful

Icons

Fontawsome

Great looking simple icons that you can integrate with your web app.

Image Gallery

Photoswipe

A simple great looking web app enabled photo gallery

Maps

jquery-ui-map

A full api that allows you to easily integrate mapping features into your web app.

Making it native

PhoneGap

If you reach the point you need to go native (phone apps of course) then PhoneGap can be a great way to go.
